The project is about to build a simple recipe website, and this website will consist of a main index page which will have links to a few recipes. In this project, I will demonstrate how to create a website with HTML. I will make use of the following skills:
    Implementing basic HTML structure and syntax such as element and tag, boilerplate
    Working with text, lists, images, and links
    Using Git and GitHub
    Commiting message with git
    creating and linking multiple web pages
